Tentang Saya

Network Engineer

Pengalaman saya dalam melakukan konfigurasi jaringan antara lain WAN, LAN, dan PAN.

Saya menguasai beberapa hardware seperti Mikrotik dan Ubiquiti

Bekerja di PT.GMI (Generasi Multi Indonusa) sebagai seorang Network Engineer membuat saya menguasai banyak hal baru serta pengalaman kerja di dunia industri.

Web Developer
